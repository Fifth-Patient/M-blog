# myBlog

一个使用express和mongodb搭建的博客demo

## 目录说明
```
│ models        操作数据库文件
|
│ public        静态文件，样式或图片
|   ├css
|   └─img
|
│ routes        路由
|   index.js
|
│ views         模版文件
|
│ index.js      项目入口文件
| package.json
```


#### eslint配置说明
- 使用`standard`规范
- 使用es6模式解析
- 设置源码类型为module
- 出现多余的分号警告
- 环境配置：`node`, `es6`, `mongo`

### view 组件
- layout
- nav*
- nav-setting*
- nitification*
- post-conent
- comments
